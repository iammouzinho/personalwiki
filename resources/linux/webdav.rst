Webdav
======

Upload a file


.. code-block:: bash

    $ sudo apt-get install cadaver
    $ cadaver https://example.com/webdav
        Username:
        Password:
    $ dav:/webdav/> put /path/of/file/on/local/to/upload/to/remote/server
